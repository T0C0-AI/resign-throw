# 🗑️ 사직서 던지기 (Resign Throw)

> 부장님 얼굴에 사직서를 휙! — 직장인의 로망을 담은 1분짜리 스트레스 해소 웹게임

[![라이브 데모](https://img.shields.io/badge/▶_라이브_데모-지금_바로_플레이-c44?style=for-the-badge)](https://t0c0-ai.github.io/resign-throw/)

**👉 지금 바로 플레이: https://t0c0-ai.github.io/resign-throw/**

설치도, 회원가입도, 다운로드도 필요 없어요. 위 링크만 누르면 바로 던질 수 있습니다. 📱 폰에서도 잘 돌아갑니다.

---

## 🎮 어떤 게임인가요?

사직서를 손가락으로 **잡고 → 휙 던져서** 칙칙한 사무실의 부장님 얼굴에 명중시키는 게임입니다.

- 부장님은 사직서가 날아오면 **슬쩍 피합니다** 😏 (그래서 조준이 중요해요)
- 정통으로 맞추면 부장님이 **"으악!"** 하고 비명을 지릅니다
- 사직서 **10장**을 다 던지면 오늘의 사직 성과가 정산됩니다
- 결과는 **단톡방에 자랑**할 수 있게 복사 버튼 한 번이면 끝

## ✨ 특징

| 재미 요소 | 설명 |
| --- | --- |
| 🎯 **조준의 묘미** | 부장님이 좌우로 왔다갔다 + 사직서를 피해서, 맞추려면 타이밍이 필요 |
| 🔥 **콤보 시스템** | 연속으로 맞출수록 점수 보너스가 쌓임 |
| 💥 **퍼펙트 명중** | 얼굴 정중앙에 맞으면 슬로우모션 + 플래시 연출 |
| 🗯️ **약올리는 대사** | 빗나가면 부장님이 "메~롱 😝", "안 받아줄 건데~?" 약을 올림 |
| 🔊 **효과음** | 던지는 소리·퍽 소리·만화풍 비명까지 — 전부 코드로 합성 (외부 음원 파일 0개) |
| 📋 **결과 공유** | 명중률·최고 콤보·총점을 한 줄 텍스트로 복사 |

## 🕹️ 조작법

1. 화면 아래쪽 **사직서를 손가락(또는 마우스)으로 잡습니다**
2. 부장님을 향해 **위로 휙 던집니다**
3. 끝! (아래로 휙 하면 발사 안 됨 — 실수 방지용)

## 🛠️ 기술 메모

- **순수 HTML 파일 1개** — 프레임워크·빌드 도구·외부 라이브러리 전부 없음
- 그래픽은 전부 **CSS** (이미지 파일 0개, 파비콘도 인라인 SVG)
- 물리(포물선·중력·공기저항)는 `requestAnimationFrame` 게임 루프로 직접 계산
- 효과음은 **Web Audio API**로 실시간 합성
- 그래서 **`index.html` 더블클릭만으로도** 인터넷 없이 바로 실행됩니다
